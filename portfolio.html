Portfolio

I built this portfolio using GitHub, which let me control every part of the website. For the bubbles on the homepage, I used a tool called Three.js. This adds a fun, moving element that catches the eye right away. To make sure the website works well for everyone, I asked many people to test it and give me their thoughts. I made changes based on what they said, which helped make the site easier to use and more enjoyable. While working on this project, I also learned how to use Artificial Intelligence (AI) as a helper. AI tools helped me find and fix problems in my code more quickly. They also gave me new ideas for designing the site.
