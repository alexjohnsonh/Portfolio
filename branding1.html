Colour

I created a brand guide and a website for Colour. The website aims to empower designers, colour enthusiasts, and curious individuals with the knowledge and tools to make more informed and intentional colour choices. By providing a comprehensive and centralised resource, I hope to foster a deeper appreciation and understanding of colour's role in our lives and work, leading to more effective communication and improved design outcomes. This website was created by using Github and Visual Studio Code.
